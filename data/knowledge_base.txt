PRODUCT OVERVIEW
----------------

Seliq Analytics is a cloud-native Business Intelligence (BI) platform built specifically for European small and medium-sized enterprises (SMEs). Founded in 2021 and headquartered in Amsterdam, Seliq Analytics empowers growing businesses to transform raw operational data into actionable insights — without requiring a dedicated data engineering team.

The platform offers an intuitive drag-and-drop dashboard builder, automated report scheduling, real-time data sync, and AI-assisted analysis powered by natural language queries. Customers can ask questions like "What were our top five products by revenue last quarter?" and receive instant visual answers drawn directly from their connected data sources.

Seliq Analytics is designed with European data privacy requirements at its core. All data remains within EU borders (AWS eu-west-1, Dublin, Ireland), and the platform is fully GDPR-compliant out of the box. Customers retain full ownership of their data at all times, and Seliq Analytics will never monetise, share, or use customer data for any purpose other than delivering the contracted service.

Key platform highlights:
- No-code dashboard creation for non-technical users
- AI-powered natural language query interface ("Ask Seliq")
- Automated anomaly detection and alerting
- Scheduled report delivery via email, Slack, or webhook
- Role-based access control (RBAC) with granular permissions
- White-label options for agencies and resellers
- Full REST API for custom integrations and data exports
- Mobile-responsive dashboards accessible on any device

COMMON ISSUES & RESOLUTIONS
------------------------------

ISSUE 1: Dashboard not loading / blank screen
Symptom: User opens a dashboard and sees a blank page or spinning loader that never resolves.
Cause: Typically caused by a cached session conflict, an expired authentication token, or a browser extension interfering with JavaScript.
Resolution:
1. Hard refresh the browser (Ctrl+Shift+R on Windows/Linux, Cmd+Shift+R on Mac).
2. Clear browser cache and cookies for seliq.io.
3. Try an incognito/private browsing window to rule out extension conflicts.
4. If the issue persists, log out, wait 60 seconds, and log back in.
5. If still unresolved, collect the browser console error log (F12 > Console) and submit it to support@seliq.io.

ISSUE 2: Data source sync stopped / data appears stale
Symptom: Dashboard data is not updating; "Last synced" timestamp is several hours or days old.
Cause: The connection credentials for the data source have expired, the source API changed its authentication requirements, or the source has rate-limited Seliq's connector.
Resolution:
1. Navigate to Settings > Data Sources > [Source Name] > Edit Connection.
2. Re-authenticate by clicking "Reconnect" and following the OAuth flow or re-entering API credentials.
3. Click "Sync Now" to trigger an immediate sync.
4. If the sync fails, check the error message displayed on the data source card. Common errors: "401 Unauthorized" (credentials expired), "429 Too Many Requests" (rate limit — retry in 1 hour), "Connection refused" (source server may be down).
5. Contact support if the issue persists after re-authentication.

ISSUE 3: API key not working / 401 errors
Symptom: Calls to the Seliq REST API return 401 Unauthorized.
Cause: The API key has been rotated, the wrong key is being used, or the key lacks the required permissions.
Resolution:
1. Go to Settings > API & Integrations > API Keys.
2. Verify the key being used matches what is displayed (only the last 4 characters are shown for security — generate a new key if unsure).
3. Ensure the API key has the required scopes for the endpoint being called (check API documentation at api.seliq.io/docs).
4. Regenerate the API key if necessary (old key is immediately invalidated upon regeneration).
5. For webhook or Zapier integrations, update the key in the connected service after regeneration.

ISSUE 4: Password reset email not arriving
Symptom: User requested a password reset but has not received the email.
Cause: Email delivery delay, spam filter, or email address not associated with a Seliq account.
Resolution:
1. Check the spam/junk folder and any email filtering rules.
2. Wait up to 10 minutes — delivery can be delayed during high-traffic periods.
3. Verify the email address is the one registered with Seliq (the reset will silently fail if the address does not match any account, as a security measure).
4. If still not received after 15 minutes, contact support@seliq.io with the email address used, and support can manually trigger the reset or verify account ownership.

ISSUE 5: Mobile app crashes on launch
Symptom: The Seliq mobile app (iOS or Android) crashes immediately on launch or after logging in.
Cause: Outdated app version, corrupted local cache, or OS compatibility issue.
Resolution:
1. Update the Seliq app to the latest version from the App Store or Google Play.
2. If already on the latest version, force-close the app and reopen.
3. Clear the app cache: iOS (Settings > Seliq > Clear Cache), Android (Settings > Apps > Seliq > Storage > Clear Cache).
4. Uninstall and reinstall the app as a last resort (local preferences will be cleared but account data is unaffected).
5. Minimum supported versions: iOS 15 / Android 10. Older OS versions are not supported.

CANCELLATION POLICY
---------------------

Customers may cancel their Seliq Analytics subscription at any time from Settings > Billing > Cancel Subscription.

Monthly plans: Cancellation takes effect at the end of the current billing period. No refunds are issued for partial months.

Annual plans: Cancellation takes effect at the end of the annual billing period. Refunds for unused months are available within the first 30 days of purchase (pro-rated). After 30 days, no refunds are issued for the remaining term, but access continues until the period ends.

Data retention after cancellation: Customer data is retained for 60 days after cancellation to allow for re-subscription or data export. After 60 days, all data is permanently deleted from Seliq's systems.

To request a data export before cancellation, use the Export All Data function under Settings > Data Management, or contact support@seliq.io.
